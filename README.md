# PasswordChecker
A Python app to check if your password is in the Rockyou data leak. You need to manually download rockyou.txt and move it to the same folder as the Python file in order to make it work.

**HOW TO RUN**
1) download the file
2) open a terminal in the folder where you have saved the file. Make sure that rockyou.txt is in the same folder.
3) run the python file
